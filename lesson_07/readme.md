## Обчислення римських чисел

Візьмемо число `IV` (4):

- `I` = 1
- `V` = 5

Коли `I` стоїть перед `V`, це означає "1 до 5" тобто 5-1 = 4.
Тому при `I` < `V`, ми віднімаємо `I` від `V`

Візьмемо число `VI` (6):

- `V` = 5
- `I` = 1

Коли `I` стоїть після `V`, це означає "5 плюс 1" тобто 5+1 = 6.
Тому при `V` > `I`, ми додаємо `V` до `I`
